# Typing-speed
About System
The project file contains python scripts (Typingspeed.py). This app project just contains the user section. The player/user can improve their Typing speed playing this game. You have to type as many words as you can in the given amount of seconds to score. Otherwise, you will lose the game, and to play again, you just have to type the current word. Also, your score will be reset and the high score will be set on the game. The design of this project is pretty simple so that the user wonâ€™t find any difficulties while working on it.

How To Run The Project?
To run this project, you must have installed Python on your PC. After downloading the project, you have to follow the steps below:

Step1: Extract/Unzip the file

Step2: Go inside the project folder, open cmd then type Typingspeed.py and enter to start the system.

OR

Step2: Simply, double click the Typingspeed.py file and you are ready to go.
